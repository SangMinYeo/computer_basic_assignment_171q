# 퀸 (밴드)
> 퀸(Queen)은 영국의 록 밴드이다. 1970년대 중반부터 1990년대 초반까지 주로 활동하였으며, 현재까지 세계적으로 많은 팬을 보유하고 있다. 국가별 판매량 인증 합산에 따르면 퀸은 세계적으로 통산 약 9천6백만 장의 음반을 판매한 것으로 집계되었으며(역대 12위), 비공식 기록까지 합산한 추정 판매량은 1억5천만~2억 장에 이른다.
